# book-catalog
List of books and authors

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
## General info
We use:
* Vue - JavaScript Framework
* Airtable - database
